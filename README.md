# PROJECT MONSTER

Popis: Webová prezentace společnosti energetických nápojů Monster. Stránka je one-page s několika sekcemi, které ukazuje produkty, aktuální příchutě a ostatní podrobnosti

Cílová skupina: Děti od 15+

Návrh obsahových sekcí: Domů, Produkty, O nás, Kontakty

Návrh designu: Hlavní barvy - Zelená a Černá. Pozadí bude černé a texty budou zelené/bílé.
