# PROJECT MONSTER

**Žák:** Tomáš Třísko  
**Téma:** One-page webová prezentace značky Monster Energy

## Stručný popis tématu
Projekt se zaměřuje na tvorbu jednostránkové (one-page) webové prezentace pro značku energetických nápojů Monster Energy. Cílem je vytvořit vizuálně atraktivní web, který představuje portfolio produktů a image značky. Technickým jádrem projektu je práce s daty – sekce s příchutěmi (produkty) se nebude vypisovat ručně, ale bude se dynamicky načítat z externího JSON souboru pomocí technologie AJAX.

## Cílová skupina
* Teenageři a mladí dospělí (15+)
* Hráči počítačových her (gamers)
* Fanoušci extrémních sportů a aktivního životního stylu

## Návrh obsahových sekcí
1.  **Domů (Hero sekce):** Úvodní část s hlavním sloganem, logem a výrazným vizuálem.
2.  **Produkty:** Přehled aktuálních příchutí a bestsellerů. Tato sekce je generována automaticky pomocí JavaScriptu (načítání dat).
3.  **O nás / Lifestyle:** Informace o značce, propojení s komunitou a sportem.
4.  **Kontakty:** Patička s odkazy na sociální sítě a copyrightem.

## Návrh designu / Wireframe
* **Barevné schéma:** Temný design (Dark Mode). Pozadí tvoří odstíny černé (`#0d0d0d`) v kontrastu s ikonickou neonově zelenou barvou Monster (`#97D700`) a bílým textem.
* **Typografie:** Výrazné, ostré bezpatkové písmo (font *Oswald*) pro nadpisy.
* **Rozložení:** Responzivní design, který se přizpůsobuje velikosti zařízení (na mobilu řazení pod sebe, na PC vedle sebe).

## Použité technologie
* **Frontend:** HTML5, CSS3 (Flexbox, CSS Proměnné), JavaScript.
* **Data:** JSON soubor (`products.json`) sloužící jako datový zdroj.
* **AJAX:** Asynchronní načítání dat o produktech (Fetch API) bez nutnosti obnovovat stránku.
* **Verzování:** Git + GitHub.
